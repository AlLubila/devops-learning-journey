This project is part of my DevOps learning journey.

The goal of this project is to automate the installation and configuration of an Nginx web server on RPM-based Linux distributions using Bash scripting.

The script installs and configures:
- Nginx web server
- Firewall rules for HTTP access
- Custom web page
- Service startup and boot persistence

---

## Technologies Used

- Linux
- Bash
- Nginx
- RPM
- YUM
- DNF
- Firewalld
- Systemctl
- Git
